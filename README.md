# 🚀 Smart Recruiter System

An AI-inspired recruitment platform that analyzes resumes, matches candidate skills with job requirements, and ranks applicants using an intelligent scoring engine.

---

## 🎯 Problem Statement

Recruiters spend hours manually filtering resumes, often missing the best candidates.

---

## 💡 Solution

Smart Recruiter System automates the hiring process by:

* 📄 Collecting candidate resumes and skills
* 🧠 Matching them with job requirements
* 📊 Ranking candidates based on relevance

---

## ⚙️ Features

* Candidate profile creation
* Resume upload
* Skill-based filtering
* Smart candidate ranking
* Match percentage system

---

## 🧠 How It Works

1. Candidates submit their skills and resumes
2. Recruiters create job postings
3. System compares required vs candidate skills
4. Candidates are ranked based on match score

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript (or React)
* Backend: Node.js, Express
* Database: MongoDB
